# くさかったら「くっせえ」と言うセンサー

くさい状況を検出し、オーディオで反応を示すセンサーを作成する。センサーにはTGS2450を使用する。

## 概要

- **センサーの種類**: TGS2450
- **検出対象**: 悪臭や異常な臭い
- **反応**: 「くっせえ」という音声メッセージを出力
